# tech-defect-prediction/
# Проект анализа дефектов в программном обеспечении

## 📋 Описание проекта

ETL-конвейер и исследовательский анализ данных (EDA) для датасета предсказания дефектов ПО. 
Проект позволяет загружать данные о метриках кода с Kaggle, выполнять их очистку и трансформацию, 
а также проводить глубокий анализ для выявления факторов, влияющих на появление дефектов.

## 🎯 Ключевые возможности

- Загрузка данных с Kaggle API или из локального файла
- Полноценный ETL-конвейер (Extract, Transform, Load)
- CLI интерфейс для гибкого управления
- Валидация и очистка метрик кода
- Визуализация и анализ в Jupyter Notebook
- Подготовка данных для машинного обучения
